Ce jeu, créé dans le contexte du cours de "Jeux vidéo 2D" à l'Université de Lausanne, met en avant une série de mini-jeux, similaire à un processus d'un Warioware, pour explorer les différentes métamorphoses animales existantes.

Le public cible est celui de tout âge, n’importe qui peut jouer et comprendre le jeu, et essayer de battre son score précédent!
